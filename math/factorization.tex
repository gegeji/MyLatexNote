\section{$A=LU$}

$$
\left(E_{32} E_{31} E_{21}\right) A=U \text { becomes } A=\left(E_{21}^{-1} E_{31}^{-1} E_{32}^{-1}\right) U \quad \text { which is } \quad A=L U
$$

高斯消元法将$A$变成上三角矩阵，等价于左乘以一系列的初等矩阵。
